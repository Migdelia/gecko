# Información de contenido #

* carpetas *
- carpeta assets : repos de librerías utilziadas (repositorio completo para facilitar futuros updates)
- carpeta css : archivos CSS compilados
- carpeta fonts : archivos de fuentes (se utiliza la fuente Open Sans)
- carpeta img : imágenes que utiliza el diseño de la aplicación
- carpeta inc : archivos PHP que se incluyen en los layout y que son comúnes a varias vistas
- carpeta js : archivos JS
- carpeta locale : archivos para la traducción a distintos idiomas (por defecto español)
- carpeta sass : estilos personalizados en SCSS (styles.sass es el archivo principal que se compila en /css)


* archivos *
- index.php : vista de login a la aplicación
- home.php : vista de panel de escritorio del usaurio


* clases css *
- table-responsive: para que las tablas tengan un comportamiento adaptado de acuerdo a la resolución de pantalla
- sort-asc: para que el th de la columna de la tabla muestre una flecha de orden ascendente
- sort-desc: para que el th de la columna de la tabla muestre una flecha de orden descendente
- sort-none: para que el th de la columna de la tabla no muestre ninguna flecha de orden
